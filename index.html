DependNet  by ObjectWind 

 

DependNet is a dependency network analysis tool for Java programs.   DependNet parses Java code beginning at the main class and follows dependencies through the code while analyzing the network of dependencies.  It then generates a portion of the Martin Dependency Metrics for each package.   For a complete explanation of the Martin Metrics please see www.objectmentor.com and look at the articles on the Principles of Object Oriented Design.

DependNet analyzes the  .class files of a Java Program, not the raw source code.  This allows for a more accurate analysis.  DependNet makes use of the freely distributed CFParse library available from the IBM alphaWorks, but also embedded in dependNet.jar.
